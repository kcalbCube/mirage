# mirage
Engine for multiplayer tile-based games written in modern C++.

### libraries in use:
* [boost](https://www.boost.org/)
* [fmt](https://github.com/fmtlib/fmt)
* [fmtlog](https://github.com/MengRao/fmtlog)
* [magic enum](https://github.com/Neargye/magic_enum)
* [SDL2](https://www.libsdl.org/)
* [Taywee/args](https://github.com/Taywee/args)
* [EnTT](https://github.com/skypjack/entt)
* [SDL_gpu](https://github.com/grimfang4/sdl-gpu/blob/master/include/SDL_gpu.h)
* [imgui](https://github.com/ocornut/imgui)
